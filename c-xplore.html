Chat Room

Chat Room Documentation

This is a real-time chat room where users can send and receive messages in a specific chat room. Below are the details on how the system works.

WebSocket Endpoint

To connect to a chat room via WebSocket, use the following WebSocket URL:

wss://gopher.bicarakan.id/ws/{room_id}

Replace {room_id} with the name of the room you wish to join. For example, to join the "general" room, use:

wss://gopher.bicarakan.id/ws/general

This WebSocket connection will allow you to send and receive messages in real-time for the specified room.

Message Format (Sent via WebSocket)

When sending a message via WebSocket, the message should follow this JSON format:


{
  "user": "username",        // The name of the user sending the message
  "text": "message text",    // The content of the message
  "timestamp": "ISO timestamp",  // The timestamp when the message was sent
  "room_id": "room_id"       // The ID of the chat room
}

Message History Endpoint

To retrieve the message history for a specific room, use the following HTTP GET endpoint:

https://gopher.bicarakan.id/messages/{room_id}

Replace {room_id} with the name of the room you want to retrieve messages for. For example, to get the message history for the "general" room, use:

https://gopher.bicarakan.id/messages/general

The response will contain an array of messages for the specified room.
